1: Placeholders and Classes Font size secondary placeholderLink

Since: 2.0.0 New Issue

A placeholder to mimic secondary styles. May also be used as a class if $burf-extras is enabled. Good for captions and other secondary information.

Example:

Make a caption feel like a secondary element to standard content.
			.caption {
					@extend %font-size-secondary;
				}
