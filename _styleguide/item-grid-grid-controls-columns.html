Grid Controls Change the number of columns in the gridLink

Since: 1.2.0 New Issue

$grid-number-columns can be used to adjust the number of columns you would like to use in your grid. By default, we use a 12 column grid. You may want to adjust this up or down for more design control.

Because the grid in Responsive is dynamically generated, you can change this freely and your "friendly" classes - col-md-third, etc - will work. That means if you're mid-project and realize you need tighter control over placement of items on your page, you can switch to a 24 column grid system without any adverse effects.

To use styles with the friendly grid classes in burf-theme, you must set this to a number that is divisible by 2, 3, and 4.

Examples

Change the grid system to use a 24 column grid

		$grid-number-columns:                      24;
