Javascript

Toggle | toggle.js

Adds responsive navigation menu for small screens, expandable search form.

Markup for responsive navigation menu:

<button class="navToggle">Toggle Navigation</button>

<nav class="mainNav" role="navigation">
	<ul id="nav">
		<li><a href="#1">Link #1</a></li>
		<li><a href="#2">Link #2</a></li>
	</ul>
<nav>

Markup for expandable search form:

<button class="searchToggle">Toggle Search</button>

<form id="quicksearch" method="get" action="">
	<input name="q" type="text" id="q" placeholder="Search site...">
	<input class="button" type="submit" name="do_search" value="Search" >
<form>

Styles for this functionality live in the burf-theme.
